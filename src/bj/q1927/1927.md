## 백준 1927. 최소 힙

### 문제 설명

- 연산의 개수 n, 그리고 다음 n개의 줄에 정수 x가 주어짐
- x가 0이라면 가장 작은 값을 출력하고 그 값을 제거, 만약 null일 경우 0 출력
- x가 자연수라면 그 값을 추가

### 문제 해결 방법

- 최소 힙을 구현하기 위해 자바에서 지원하는 `PriorityQueue` 사용
- Java의 `early return` 처럼 코드를 구현하기 위해 각 조건문에 `continue` 사용
- x가 0이고 배열의 가장 작은 값이 `null` 일 경우에 `NPE`를 방지하기 위해, `int`가 아닌 `Integer` 래퍼 클래스 사용
