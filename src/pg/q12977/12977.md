## 프로그래머스 12977. 소수 만들기

### 문제 설명

- 3개 이상의 수가 배열로 주어짐
- 수가 들어있는 배열에서, 서로 다른 3개를 골라 더했을 때 소수가 되는 경우의 개수 반환

### 문제 해결 방법

- nCr 조합 구현 — 크기 n 집합의 부분집합 중 크기가 r인 것만 선택하는 방식
- DFS() 가 각 원소에 대해 선택 / 미선택 두 가지로 재귀 분기 → 이진 트리 형태로 탐색
- r개를 모두 골랐으면 합계를 구해 소수 판별
- selected.add() 로 선택 후 재귀가 끝나면 selected.remove() 로 원상복구
