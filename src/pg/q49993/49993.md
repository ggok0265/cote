## 프로그래머스 49993. 스킬트리

### 문제 설명

- 선행 스킬 순서가 담긴 문자열과 스킬트리 배열이 주어짐
- 각 스킬트리에 선행 스킬 순서에 포함된 스킬이 있다면, 그 순서대로 스킬을 배워야 함
- 선행 스킬 순서에 포함되지 않은 스킬은 순서에 상관없이 배울 수 있음
- 스킬트리 배열 중, 가능한 스킬트리의 개수를 반환

### 문제 해결 방법

- 각 스킬트리에서 선행 순서와 무관한 스킬은 검사할 필요가 없으므로, 정규식 `[^ ]`을 사용하여 제거
- 제거 후 남은 문자열이 선행 스킬 순서와 일치하는지 `startsWith`로 확인, 일치하면 카운트 증가
