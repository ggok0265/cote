## 프로그래머스 42748. K번째수

### 문제 설명

- 배열의 i번째 숫자부터 j번째 숫자까지 자르고 정렬했을 때, 그 배열의 k번째 수 찾기
- 자르고 정렬하여 수를 찾을 1차원 배열이 주어짐
- [i, j, k]를 원소로 가지는 2차원 배열이 주어짐
- 2차원 배열의 모든 원소에 대한 결과를 배열에 담아 출력해야 함

### 문제 해결 방법

- 답을 저장할 int 배열 선언, 이 배열의 크기는 2차원 배열의 크기와 같게 설정
- 빠른 정렬을 위해 자른 원소를 저장할 `ArrayList` 선언
- 2차원 배열에 주어진 i, j 값은 1부터 시작하기 때문에, 인덱스 값을 맞춰주기 위해 i-1, j-1로 연산 하여 반복
- 정렬된 `ArrayList` 에서도 k는 1부터 시작하기 때문에 List의 k-1번째 값을 정답 배열에 저장
